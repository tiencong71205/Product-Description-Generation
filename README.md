# Product-Description-Generation
Automatic product description generation uses NLP and deep learning to create meaningful descriptions from product data. A transformer-based model learns from existing data to produce consistent, scalable, and human-like content, reducing manual effort and improving efficiency in e-commerce platforms.

## 🚀 Usage

- **Step 1: Clone repository**
```bash
git clone https://github.com/tiencong71205/Product-Description-Generation.git
cd Product-Description-Generation
```

- **Step 2: Cài đặt thư viện**
```bash
pip install torch transformers sentencepiece
```

- **Step 3 : Tải model**
👉 Tải model tại đây:
https://drive.google.com/uc?id=1njBzldqd1nVXf69LUbgr9GecYEKVe31X
- Sau đó giải nén và đặt vào:
``` bash
./model
```
- **Step 4 : Chạy model**
``` bash
python run_model.py
```
